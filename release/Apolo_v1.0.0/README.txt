Solitario Apolo - Version 1.0.0

ATENCION: Se requiere una pantalla de al menos 1600x900 para poder visualizar el tablero completo.

//INSTALACION

-Es necesario tener instalado Java Runtime Environment (JRE), disponible de forma gratuita en la página web de Java para sistemas operativos Windows.

-Descargar el archivo comprimido del enlace de acceso.

-Descomprimir la carpeta en cualquier ubicación.

-Acceder a la carpeta Apolo_v1.0.0.

-Ejecutar Apolo_v1.0.0.jar

//MENU PRINCIPAL

-Jugar: Abre la ventana de inicio de juego, donde se debe introducir el nombre del jugador, y el modo de juego deseado.

-Puntajes: Este boton abre una ventana, donde se muestran los 10 mejores puntajes.

-Reglas: Se abre una ventana, donde se muestran las reglas del juego.

-Salir: Cierra el programa.
